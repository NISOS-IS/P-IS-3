# Análisis de Requisitos

**Actores:**

>**Principal:** Profesor   
>**Secundario:** Alumno

**Datos:**

* DNI
* Nombre
* Apellidos
* Teléfono
* Email
* Dirección
* Curso
* Fecha de Nacimiento
* Equipo
* Líder

**Requisitos funcionales:**

>**Prioridad: 1**

>* Buscar alumno
>* Insertar alumno
>* Mostrar listado de alumnos
>* Guardar copia de seguridad
>* Login del profesor
>* Registrar profesor

>**Prioridad: 2**

>* Mostrar alumno
>* Cargar copia de seguridad

>**Prioridad: 3**

>* Modificar alumno
>* Borrar alumno
  

**Requisitos no funcionales:**

* Programación en C++.
* Documentación en lenguaje Markdown
* Uso del sistema de control de versiones Git y la plataforma GitHub
* Fichero binario de copia de seguridad
* Máximo de alumnos: 150
* Guardar datos del listado de alumnos mostrados en archivo HTML
